## Typography

- **Headings**: Bold, larger sizes (32px, 24px, 18px)
- **Body**: 16px for main text, 14-15px for lists
- **Footer**: 12-13px for fine print
- **Font Stack**: System fonts (-apple-system, Segoe UI, Roboto, etc.)

## Responsive Design

The email is mobile-responsive:
- Maximum width: 600px (centers on desktop)
- Padding adjusts for mobile screens
- Button is touch-friendly (48px height)
- Text size is readable on small screens
